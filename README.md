## 📌 About

This repository contains the source code for my personal developer portfolio — a single-page site built with **vanilla JavaScript, HTML5, CSS3, and Firebase**, showcasing real, working projects rather than templates. It's designed to demonstrate practical frontend skills: responsive layouts, real-time data syncing, API integration, and clean UI/UX.

**🌐 Live site:** [uzairbaigdev.github.io/Portfolio-repo](https://uzairbaigdev.github.io/Portfolio-repo)

The portfolio includes six sections — **Home, About, Skills, Projects, Space, and Contact** — each built to be fast, mobile-friendly, and visually polished, with a distinct "Space Science" section reflecting a personal interest in astrophysics alongside the core dev work.

## ⚙️ Getting Started

```bash
# 1. Clone the repository
git clone https://github.com/uzairbaigdev/Portfolio-repo.git

# 2. Move into the project folder
cd Portfolio-repo

# 3. Open it locally
# Simply open index.html in your browser, or serve it:
npx serve .
```

No build step required — this is a static site, so it also deploys directly via **GitHub Pages**.
